注意事项

  • ⚠️ 本文基于 OpenClaw 2026.3.x 版本，不同版本指令可能有差异
  • ⚠️ 提权模式请谨慎使用，确保在安全环境下操作
  • ⚠️ 定期查看 Token 用量，避免 API 超额
  • ⚠️ 敏感配置（如 API 密钥）不要提交到代码仓库
