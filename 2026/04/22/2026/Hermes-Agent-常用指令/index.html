磁头笔记：Hermes-Agent 常用指令

什么是 Hermes-Agent

Hermes-Agent 是一款运行在终端的 AI 助手，可以帮你完成各种任务，从简单的文件操作到复杂的代码编写、网页交互、自动化任务等。

常用指令

1. 文件操作

2. 终端命令

3. 网页交互

4. 代码执行

5. 任务管理

6. 记忆管理

7. 技能管理

8. 会话搜索

使用技巧

  1. 多步骤任务：使用 todo 创建任务列表，按步骤完成
  2. 复杂操作：使用 execute_code 编写 Python 脚本处理批量任务
  3. 网页自动化：结合 browser_navigate + browser_click + browser_type 完成表单填写
  4. 持久化知识：使用 memory 保存重要配置和偏好设置
  5. 复用经验：将常用工作流程保存为 skill，下次直接调用

注意事项

  • 文件路径支持绝对路径和相对路径
  • 终端命令默认 180 秒超时，长任务需设置 timeout 参数
  • 网页操作需要先 browser_navigate 再执行其他操作
  • 敏感信息（如密码、token）不要保存在记忆中

本文基于实际使用经验总结，更多功能请参考官方文档。
